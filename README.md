# Métodos Matemáticos e Análise de Dados

Repositório de aulas e exercícios sobre Métodos Matemáticos aplicados à Análise de Dados, desenvolvido como parte da disciplina do 3º Período da UNIFEI.

## 📋 Conteúdo do Repositório

### 🐍 ConceitosPython
Fundamentos de Python para análise de dados.
- **Aula01**: Introdução ao Python
- **Aula02a, Aula02b**: Estruturas de dados e operações básicas
- **Aula04a, Aula04b, Aula04c**: Funções e programação avançada

### 📊 DataFrames_Pandas
Manipulação e análise de dados com Pandas.
- **Aula03**: Criação e manipulação de DataFrames
- **Aula08**: Operações avançadas com DataFrames

### 📈 PlotagemGraficos
Visualização de dados com Matplotlib, Plotly e outras bibliotecas.
- **Aula09a, Aula09b, Aula09c**: Tipos básicos de gráficos
- **Aula09-exerc**: Exercícios práticos de visualização
- **Aula10b, Aula10c**: Gráficos avançados
- **Aula10-exerc**: Exercícios complementares
- **Dados de exemplo**: apple.csv, tesla.csv, ebola.csv, CPI_2014.csv, lifeExpectancy.csv, migrants.csv

### 🔗 Crosstab
Análise de tabulação cruzada e tabelas de contingência.
- **Aula11**: Introdução a tabulação cruzada
- **Dados**: nobel_prize.csv

### 📊 AnaliseCombinatoria
Análise Combinatória e conceitos estatísticos fundamentais.

#### Núcleos de conteúdo:
- **Aula05, Aula06, Aula07**: Combinatória básica e permutações
- **Distribuição Normal** (subpasta):
  - **Aula12**: Distribuição Normal
  - **Aula12-exerc**: Exercícios práticos
- **Teorema Central do Limite** (subpasta):
  - **Aula13**: Teorema Central do Limite
  - **Aula13-exerc**: Exercícios práticos
- **Testes de Hipótese** (subpasta):
  - **Aula14**: Conceitos fundamentais de testes de hipótese
  - **Aula15**: Testes de hipótese avançados

### 📁 TrabalhoOrdenação
Projeto prático sobre algoritmos de ordenação com análise de desempenho.
- **Datasets**: Diferentes tipos de dados (aleatório, quase ordenado, ordenado)
- **Notebooks**: Análise e visualização de algoritmos de ordenação
  - **graficos_trab1.ipynb**: Gráficos de desempenho
  - **graficos_trabPrime.ipynb**: Análise visual

## 📖 Como Usar

1. **Ambiente Python**: Certifique-se de ter Python 3.7+ instalado
2. **Dependências**: Instale as bibliotecas necessárias:
   ```bash
   pip install pandas numpy matplotlib scipy plotly jupyter
   ```
3. **Abrir Notebooks**: Use Jupyter para abrir os arquivos `.ipynb`
   ```bash
   jupyter notebook
   ```

## 📝 Notas

- Os notebooks estão organizados sequencialmente por conceitos
- Recomenda-se seguir a ordem das aulas dentro de cada seção
- Cada notebook contém código, explicações e exercícios práticos
- Os datasets estão disponíveis nos diretórios indicados
